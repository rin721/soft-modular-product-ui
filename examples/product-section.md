# Example: Product Section

## Input Requirement

Generate a product introduction section for an AI-assisted video clipping feature. Audience: creators. Primary goal: explain the feature and drive usage.

## Skill Rules Used

- Layout grammar: product introduction section and media preview.
- Components: Section, Feature List, Card, Tag/Badge, Button.
- Tokens: accent, small-radius surfaces, display stats, 16:9 preview.
- Interaction: hover lift, focus ring, active scale.

## Generation Strategy

Create a focused section that can be dropped into a larger homepage. Keep the copy short and the preview original. Use a tag row to show modes or supported workflows.

## Structure Draft

1. Section header: icon, "AI clipping", beta badge.
2. Intro copy: one short sentence.
3. Left column: original 16:9 product mockup card.
4. Right column: three feature items.
5. Tag row: Highlights, Captions, Review, Export.
6. CTA row: Try clipping, Learn more.

## Token Notes

- Section title: `16px / 700` for compact app-style section header.
- Product title inside section: display stack at `28px-32px`.
- Tags: pill background, checked accent state.
- Preview card: `6px` radius, hover lift.

## Quality Checks

- Section can stand alone.
- No copied screenshots or original source-site media.
- Tags wrap on mobile.
- Buttons meet `44px` mobile touch target.
- Focus and disabled states are documented.
